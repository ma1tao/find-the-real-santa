# 🎅 寻找圣诞老人 (Find the Real Santa)

在这个充满节日氛围的趣味小游戏中，你需要练就一双火眼金睛，从茫茫多的“鳌拜”中找出真正的圣诞老人！

## 🎮 游戏玩法

1.  **观察**：屏幕上会出现许多个红帽子白胡子的老爷爷头像。
2.  **分辨**：仔细看！大部分是徐锦江饰演的“鳌拜”（中国版圣诞老人），只有一个是真正的西方圣诞老人。
3.  **点击**：找出真正的圣诞老人并点击他。
4.  **升级**：每找对一次，难度就会升级！网格会变得更密，干扰项也会更多。
5.  **挑战**：挑战高分纪录！如果不小心点到了鳌拜，游戏就结束啦！

## 🏆 评分规则

想拿高分？拼手速的时候到了！

*   **基础分**：每通过一关获得 **1000 分**。
*   **时间奖励**：如果在 **10秒** 内过关，剩余时间会转化为额外奖励分！
    *   `奖励分 = (10 - 消耗时间) * 100 * 当前关卡数`
    *   关卡越高，时间奖励系数越高！
*   **超时惩罚**：如果超过 10秒 还没找到，基础分会随时间衰减。

## ✨ 游戏特色

*   **魔性设定**：玩梗“红帽子白胡子老人”，让鳌拜与圣诞老人同台竞技。
*   **无尽模式**：从 1x1 的热身关卡开始，难度逐级递增，挑战你的眼力和极限。
*   **动态音乐**：经典的圣诞背景音乐（We Wish You a Merry Christmas），**随关卡推进速度会自动加快**（每关增加 2%），营造紧张刺激的氛围！
*   **欢乐音效**：完全使用 Web Audio API 生成的实时音效，无需加载音频文件。
*   **节日氛围**：精美的圣诞主题界面，飘落的雪花，闪烁的彩灯，沉浸感十足。
*   **彩蛋语音**：点错时会有惊喜提示：“金钩拜，金钩拜，金钩鳌了拜！”

## 🚀 如何运行

**前提条件：** Node.js 环境

1.  **安装依赖**：
    ```bash
    npm install
    ```

2.  **启动游戏**：
    ```bash
    npm run dev
    ```

3.  **开始游玩**：
    打开浏览器访问 `http://localhost:3000` 即可开始挑战！

## 🛠️ 技术栈

*   React 19
*   TypeScript
*   Vite
*   Tailwind CSS
*   Web Audio API (用于生成音乐和音效)
